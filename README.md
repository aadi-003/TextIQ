# 💡 TextIQ

TextIQ is a Python-based desktop NLP application built using Tkinter. It integrates multiple APIs to perform various Natural Language Processing (NLP) tasks through an easy-to-use graphical interface.

---

## ✨ Features

- 🔐 User Registration & Login
- 🌍 Language Detection
- 😊 Sentiment Analysis
- 🔍 Text Similarity Checker
- 💾 JSON-based local database
- 🎨 Simple and user-friendly Tkinter GUI

---

## 🛠️ Tech Stack

- Python
- Tkinter
- REST APIs
- JSON
- python-dotenv
- Git & GitHub

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/aadi-003/TextIQ.git
```

Go into the project

```bash
cd TextIQ
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file using `.env.example`

Run the application

```bash
python main.py
```

---

## 🔑 Environment Variables
